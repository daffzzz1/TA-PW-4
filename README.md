Sistem ini adalah aplikasi sederhana berbasis PHP yang digunakan untuk mengelola data kontak dengan fitur CRUD (Create, Read, Update, Delete). Aplikasi ini menggunakan PHP Session sebagai penyimpanan sementara, sehingga cocok untuk pembelajaran konsep dasar CRUD, validasi form, dan manajemen data sederhana tanpa database.
